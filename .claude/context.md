# Contexte Projet - PolitiqueAi

Plateforme SaaS d’évaluation de discours politiques.

Langue : Français strict
Cible : Étudiants, chercheurs, analystes politiques

Fonctionnalités clés :
- Analyse IA
- Reformulation
- Gestion abonnements
- Interface admin complète

Stack :
- Next.js 14
- Supabase
- Tailwind
- Vercel

Contraintes :
- UX simple
- Responsive total
- Sécurité forte
